# python-assignment_4
***숫자 야구 게임***

**<규칙>**
> 사용자는 'Enter'키를 통해 게임을 시작한다.
> 
> 컴퓨터가 랜덤으로 0부터 9까지의 숫자 중 서로 다른 3개의 숫자를 골라 숫자를 만든다.
> 
> 사용자로부터 세 자리 숫자를 입력받아 컴퓨터가 랜덤 제작한 숫자와 비교한다.
> 
> 숫자는 맞지만 위치가 틀린 경우 -> BALL
> 
> 숫자와 위치가 전부 맞는 경우 -> STRIKE
> 
> 숫자와 위치가 전부 틀리면 -> OUT
> 
> 몇 번째 자리 숫자가 BALL이고 STRIKE는 알려주지 않는다.
